Overview

Find. Save. Make.

As part of a software engineering class at UCSD, I and a team of 10 other students created wateryoucooking, a recipe manager using vanilla web technologies to practice Agile engineering workflows and management.

Features

We implemented all the useful features you would expect from an online recipe manager:

Finding New Recipes

You can filter search results by dietary restrictions, cuisine type, and recipe parameters like time to cook.

Saving Recipes

From recipes you've searched or viewed, you can save a recipe into your recipebook and access it later.

Creating new Recipes

You can create custom recipes and add ingredients, steps, nutrition facts, and other details to the custom recipe.

Build Pipeline and Software Engineering Process

As part of the management and testing team, my responsibilities included setting up the GitHub Actions workflows, writing tests, and managing everyone's tasks to make sure no one was overworked.

My Responsibilities

  1. Setting up and Managing the GitHub Actions Pipeline
  2. Managing the Issues tab and assignments
  3. Organizing Meetings
  4. Writing tests
  5. Miscellaneous code contributions
